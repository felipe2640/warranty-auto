# Guia Rápido (1 página) - Garantias por Etapas

Use este resumo no início do turno. Ele serve como "cola" do dia a dia.

## 1) Fluxo das etapas (sempre nessa ordem)
Recebimento -> Interno -> Logística -> Cobrança -> Resolução -> Encerrado

## 2) Checklist rápido para avançar etapa
- Interno -> Logística: "Fornecedor definido"
- Logística -> Cobrança: "Anexo CANHOTO"
- Cobrança -> Resolução: "Resposta do fornecedor"
- Resolução -> Encerrado: "Resultado final (Crédito/Troca/Negou)"

## 3) Rotina diária recomendada (5 passos)
1) Dashboard: ver "Ações de Hoje" e "Atrasadas".
2) Agenda: trabalhar nas abas "Hoje" / "Atrasadas" / "7 Dias".
3) Garantias: buscar tickets e abrir o detalhe.
4) Ticket: usar "Registro", "Anexar" e "Avançar etapa".
5) Fechamento: registrar "Resposta do fornecedor" e "Resultado final".

## 4) Onde cadastrar no Admin
- Admin -> "Usuários" -> "Novo Usuário"
- Admin -> "Lojas" -> "Nova Loja"
- Admin -> "Fornecedores" -> "Novo Fornecedor"
- Admin -> "Configurações" -> "ID da Pasta Raiz" + "Testar Acesso ao Drive"

## 5) Atalhos úteis
- Criar ticket: "Nova Garantia" (Dashboard ou Garantias).
- Timeline: botão "Registro" (ou "Adicionar" na aba Timeline).
- Agenda: botão "Calendário" para ver o mês.
- Contato: botão "Contato" aparece apenas para Cobrança e Administrador.
