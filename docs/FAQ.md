# FAQ - Dúvidas comuns

Se a dúvida for operacional, consulte primeiro `docs/CLIENTE_MANUAL.md`.
Se for algo rápido, use `docs/CLIENTE_RAPIDO_1_PAGINA.md` ou a ajuda do sistema em `/t/<tenant>/help`.

## Etapas e regras
### "Não consigo avançar etapa"
- Clique em "Avançar etapa" e verifique a lista de "Requisitos".
- Se aparecer "Fornecedor definido", use "Definir fornecedor".
- Se aparecer "Anexo CANHOTO", use "Anexar" e selecione a categoria "Canhoto".
- Se aparecer "Resposta do fornecedor", preencha no diálogo de avançar.
- Se aparecer "Resultado final (Crédito/Troca/Negou)", selecione o resultado.

## Agenda e datas
### "A data da próxima ação ficou diferente"
- A data e salva como data (sem horário).
- Se ficou incorreta, abra "Marcar Contato Feito" ou "Adicionar Registro" e salve novamente a "Data da próxima ação".

### "Não consigo ver o botão Contato na Agenda"
- O botão "Contato" aparece apenas para Cobrança e Administrador.

## Drive e anexos
### "Erro no Testar Acesso ao Drive"
- Confirme o "ID da Pasta Raiz" em Admin -> "Configurações".
- Compartilhe a pasta do Drive com a service account do Google Drive.

### "Anexo não abre"
- Use o ícone de download para baixar o arquivo.
- Verifique se o ticket pertence ao mesmo tenant e se você tem permissão.

## Permissões
### "Não aparece Admin no menu"
- O menu "Admin" so aparece para o papel "Administrador".
- Solicite ao Admin para ajustar seu "Papel".
